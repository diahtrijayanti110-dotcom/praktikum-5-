<h2><i class="bi bi-pencil-square"></i> Tugas Praktikum</h2>

<p>
Program berikut adalah aplikasi sederhana untuk menampilkan daftar nilai mahasiswa menggunakan <b>Dictionary</b>.  
Fitur program:
</p>

<ul>
  <li><i class="bi bi-plus-circle"></i> Tambah Data</li>
  <li><i class="bi bi-pencil"></i> Ubah Data</li>
  <li><i class="bi bi-trash"></i> Hapus Data</li>
  <li><i class="bi bi-table"></i> Tampilkan Data</li>
  <li><i class="bi bi-search"></i> Cari Data</li>
</ul>

<p>
Nilai akhir dihitung dari:
</p>

<pre>
Tugas : 30%
UTS   : 35%
UAS   : 35%
</pre>
